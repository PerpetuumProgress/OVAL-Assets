Class TrafficSinkAction

Applied Stereotypes:
XSDcomplexType

Defines a sink of traffic at a specific position.

Used in:
TrafficAction
Properties
Name Type Cardinality AppliedStereotypes Description
radius double 1..1 XSDattribute Defines the radius of the traffic sink where vehicles disappear around the specified position. Unit: m; Range: [0..inf[.
rate double 0..1 XSDattribute Defines the rate on which vehicles disappear at the sinks location. If omitted, rate is interpreted as 'inf'. Unit: vehicles/s Range: [0..inf[.
position Position 1..1 XSDelement Defines the position of the traffic sink.
trafficDefinition TrafficDefinition 0..1 XSDelement Defines the vehicle and controller distribution for the sink.
